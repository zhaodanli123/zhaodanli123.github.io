简而言之，如何去找到所需要的指令呢？（ 以打印为例lp[line print] ）

  1. man lp ，找到相关说明
  2. lp[tab][tab] 找到以lp开头的指令
  3. 找到相关指令后， –help 查基本语法
  4. 若有需要再以man去查询指令的用法

TIPS：如果你知道某个指令，但却忘记了相关参数与选项，请先善用–help 来查询相关信息；当有任何你不知道的指令或文件格式等，但你想要了解它，请使用man/info查询
